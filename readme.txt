== Description ==

Kiyono is my 3rd WordPress theme named after my grandmother. Kiyono is a simple blog theme, yet it is equipped with five block patterns and a blank template for those who are experimenting with WordPressâ€™s Latest Full Site Editing.  Kiyono theme is responsive and Gutenberg ready, and it is an accessibility-ready theme. It helps to create inclusive websites and improve the user experience for those who rely on assistive technologies.
 
== Browser Support ==

Kiyono theme supports all moderns brwosers that support HTML5 and CSS custom variables. 
WordPress 5.8 will remove Internet Explorer 11 and older. 



== Theme Options ==

Kiyono offers the following options. 

1. Choose big or normal text size for the site name on desktop.
1. Show or hide the search icon in the header on desktop.
2. Show a placeholder image for the posts missing a featured image.
3. Show or hide the feature image on a single post. 
4. Keep or remove animation on feature images for better accessibility.


== Where to find the theme options ==

1. Appearance < Customize < Theme options



== Blank template == 

The blank template is a blank template so that you can freely create your own layout using the Full Site Editor. To use the totally blank template

1. Install and activate Gutenberg Plugin https://wordpress.org/plugins/gutenberg/
2. Create a new page or edit an existing page
3. Open the Template tab on the right side of the editor. 
4. Select the totally blank template from the dropdown.

If you like to add back the header and footer, select the Default template from the dropdown in the template tab on the righthand sidebar. 

More detail on the Full Site Editing
https://make.wordpress.org/design/handbook/focuses/full-site-editing/
